# 🚀 Microserviço Promoções Relâmpago

Gera um card com promoção, envia para Telegram e WhatsApp (MyZap) e roda no IIS.

## 📖 Pré-requisitos

✅ Python 3.9+  
✅ Node.js + npm  
✅ IIS + FastCGI  
✅ Bot no Telegram + Token  
✅ WhatsApp autenticado no MyZap
